Cheesy name, I know. I was on the spot when uploading the app to the server and I had to find a name which wasn't already taken. Maybe I'll change it someday.
The goal of this project is to learn to build an application from end to end using Python, Flask, HTML, CSS, JavaScript, PostgreSQL, SQLAlchemy ORM, MVx design principles and other concepts. I'm trying to be pretty thorough in documenting the process; check out the release log if you're curious. Feel free to email me if you have questions or would like the source code. I'm a big believer that helping others learn benefits everyone.

Features in development for this page:

  • Daskboard of account status
  • Stock game leaderboards
  • Some stats on the most active/best/worst-performing stocks of the day
  • Links to articles you might be interested in, based on the stocks you look at or own
  • Resources on how to learn more about the stock market and trading

Here are some features for other pages on this site which I'm considering developing:

  • Smart search (not just stock symbols, but also company names.)
  • Some decent design, including a logo
  • Stock graphs/charts
  • Historical stock data lookup
  • Email notifications
  • Sign in with either username or email address and password.
  • Possibility to sign in or create an account via Facebook or other social media.
  • Account settings/password management
  • Private messaging
  • "Follower" relationships
  • User avatars
  • Let me know if you have other ideas!

In the meantime, check out the stock page.
